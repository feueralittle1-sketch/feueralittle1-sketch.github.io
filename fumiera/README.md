# 芙咩拉官網（GitHub Pages 初版）

這是一個可直接放到 GitHub Pages 的靜態網站初版。

## 目前內容
- `index.html`：首頁
- `styles.css`：可愛風格樣式

## 發佈方式
最簡單的做法：
1. 建立 GitHub repo（例如 `fumiera-site`）
2. 把 `site/` 內檔案推上去
3. 在 GitHub Pages 選擇從 main branch / root 發佈，或改成 `/docs`
